## ⌨️ Stellarboards (Custom Keyboard Landing Page)

### 📌 Preview  
Stellarboards is a mock product landing page inspired by my interest in building and modding custom mechanical keyboards. It showcases a fictional custom keyboard brand with a structured, modern layout featuring product highlights, featured boards, and a signup form.

---

### 📖 About the Project  
This project simulates a real product landing page for a custom keyboard company called **Stellarboards**. It was built using HTML and CSS to practice structuring a multi-section website, working with navigation links, embedding media, and designing a visually organized product showcase.

The idea was inspired by the custom keyboard hobby—where enthusiasts build, modify, and personalize keyboards for performance and aesthetics. Banner and assets created from scratch!

---

### 🚀 What I Learned  
- How to structure a **multi-section landing page**  
- Using **navigation anchors** for smooth page navigation  
- Embedding a **YouTube video iframe**  
- Creating reusable UI sections like feature cards and product cards  
- Working with **forms** for email collection  
- Improving layout using **Flexbox and positioning techniques**  
- Styling a full webpage using external CSS  

---

### 🛠️ Technologies Used  
- HTML5  
- CSS3  
- Flexbox  
- Media Queries  
- YouTube Embed (iframe)
- Paint.net

---

### 🎯 Key Takeaway  
This project helped me understand how real product landing pages are structured and how HTML and CSS work together to create a complete, interactive website experience.
